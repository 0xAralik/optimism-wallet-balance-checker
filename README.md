# Optimism Wallet Balance Checker

This project is a simple Node.js script that checks an Ethereum wallet's ETH balance on the Optimism network.

## Why Optimism?
Optimism is a leading Ethereum Layer 2 scaling solution.  
This project demonstrates how to read on-chain data using Optimism RPC.

## Tech Stack
- Node.js
- ethers.js
- Optimism Mainnet

## How to Run
```bash
npm install
node index.js
